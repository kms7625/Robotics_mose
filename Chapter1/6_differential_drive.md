# 차동 구동 방식(Differential Drive) 조사 보고서

---

## 1. 라디안(Radian)

**라디안**은 각도를 나타내는 SI 단위이다.

- 원의 반지름과 같은 길이의 호(Arc)를 이루는 각도 = 1 라디안
- 원 한 바퀴(360°) = 2π 라디안

### 단위 변환

| 변환 방향 | 공식 |
|-----------|------|
| 도(°) → 라디안 | `라디안 = 도 × (π / 180)` |
| 라디안 → 도(°) | `도 = 라디안 × (180 / π)` |

**자주 쓰는 값:**

| 도(°) | 라디안 |
|-------|--------|
| 30° | π/6 ≈ 0.524 |
| 45° | π/4 ≈ 0.785 |
| 90° | π/2 ≈ 1.571 |
| 180° | π ≈ 3.142 |
| 360° | 2π ≈ 6.283 |

## 2. 차동 구동 방식의 개념

**차동 구동(Differential Drive)** 이란 좌우 두 바퀴의 **속도 차이**로 방향을 제어하는 구동 방식이다.

- 별도의 조향 장치(핸들, 서보) 없이 두 모터만으로 전진·회전·제자리 회전 가능
- 구조가 단순하여 소형 로봇에 널리 사용
- 라인트레이서 로봇의 대표적인 구동 방식

```
     [좌측 모터] ←─── 독립 제어
          │
     ┌────┴────┐
     │  로봇   │
     └────┬────┘
          │
     [우측 모터] ←─── 독립 제어
```

---

## 3. 각속도와 회전 반경

### 3.1 각속도 (Angular Velocity, ω)

단위 시간당 회전하는 각도.

```
ω = Δθ / Δt   (단위: rad/s)
```

선속도(v)와의 관계:
```
v = ω × r    (r: 바퀴 반지름)
ω = v / r
```

### 3.2 회전 반경 (Turning Radius, R)

로봇이 곡선을 그릴 때 회전 중심으로부터의 거리.

- R이 크다 → 완만한 곡선
- R이 작다 → 급격한 회전
- R = 0 → 제자리 회전

---

## 4. 각 바퀴의 각속도가 이동 경로에 미치는 영향

| 상황 | 조건 | 결과 |
|------|------|------|
| 직진 | 좌속도 = 우속도 | 직선 이동 |
| 우회전 | 좌속도 > 우속도 | 오른쪽으로 곡선 이동 |
| 좌회전 | 우속도 > 좌속도 | 왼쪽으로 곡선 이동 |
| 제자리 우회전 | 좌속도 = +v, 우속도 = -v | 중심 기준 우회전 |
| 제자리 좌회전 | 좌속도 = -v, 우속도 = +v | 중심 기준 좌회전 |
| 정지 | 좌속도 = 우속도 = 0 | 정지 |

---

## 5. 두 바퀴 속도 차이에 따른 회전 반경 및 방향 변화

좌바퀴 선속도 `v_L`, 우바퀴 선속도 `v_R`, 휠베이스(두 바퀴 간 거리) `L`

```
회전 반경 R = L/2 × (v_L + v_R) / (v_L - v_R)

로봇 중심 선속도 v = (v_L + v_R) / 2

로봇 각속도 ω = (v_L - v_R) / L
```

**예시:** v_L = 20cm/s, v_R = 10cm/s, L = 20cm
```
R = 10 × (20 + 10) / (20 - 10) = 10 × 3 = 30cm
ω = (20 - 10) / 20 = 0.5 rad/s
```

---

## 6. 제자리 회전 방법

두 바퀴를 **반대 방향, 같은 속도**로 회전시킨다.

```
v_L = +v,  v_R = -v  →  우측으로 제자리 회전
v_L = -v,  v_R = +v  →  좌측으로 제자리 회전
```

이 경우 R = 0 (두 바퀴 중간 지점이 회전 중심)

**제자리 회전 각속도:**
```
ω = (v_L - v_R) / L = 2v / L
```

---

## 7. 휠베이스(Wheel Base)가 미치는 영향

휠베이스 L = 두 바퀴 사이의 거리

| L이 클 때 | L이 작을 때 |
|-----------|-------------|
| 동일 속도 차에서 회전 반경 커짐 | 회전 반경 작아짐 (예민한 회전) |
| 직진 안정성 높음 | 좁은 공간 선회 유리 |
| 제자리 회전 시 넓은 공간 필요 | 제자리 회전 시 좁은 공간 가능 |

---

## 8. 회전 반경 계산 방법

### 엔코더 기반 계산

바퀴 둘레 `C = π × d`, PPR을 알면 1펄스당 이동 거리 계산 가능.

```
d_L = 펄스_L × (π × 바퀴지름 / PPR)   ← 좌바퀴 이동 거리
d_R = 펄스_R × (π × 바퀴지름 / PPR)   ← 우바퀴 이동 거리

회전 각도 θ (rad) = (d_L - d_R) / L
회전 반경 R       = L/2 × (d_L + d_R) / (d_L - d_R)
로봇 이동 거리    = (d_L + d_R) / 2
```

| 직진 | v_L = v_R |
| 회전 방향 | 느린 쪽 방향으로 회전 |
