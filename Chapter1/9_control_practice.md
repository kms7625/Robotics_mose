## 문제 1 — 센서값 000 → 100 변화 시 제어 절차

### 상황 분석

3개 센서를 [S_좌, S_중, S_우]로 정의한다.

| 상태         | 센서값  | 의미                                            |
| ------------ | ------- | ----------------------------------------------- |
| 정상 주행 중 | `000` | 세 센서 모두 경로(검은 선) 위에 있음            |
| 이상 감지    | `100` | 좌측 센서가 흰색(경로 이탈), 중·우측은 경로 위 |

**판단:** 좌측 센서가 경로를 벗어났으므로 로봇이 **오른쪽으로 치우쳤다.**
→ 경로를 유지하려면 **왼쪽으로 방향을 틀어야 한다.**

---

### 제어 절차 (단계별)

#### STEP 1. 센서값 읽기

```
S_좌 = 1 (흰색 감지)
S_중 = 0 (검은색 감지)
S_우 = 0 (검은색 감지)
→ 센서 패턴: 100
```

#### STEP 2. 오차(Error) 계산

```
정상(중앙) = 0, 오른쪽 치우침 = +, 왼쪽 치우침 = -

현재 상태: 좌측이 경로 이탈 → 로봇이 오른쪽으로 벗어남
오차 e = +1 (오른쪽 치우침)
```

#### STEP 3. PID 제어값 계산

```
u = Kp × e + Ki × ∫e + Kd × Δe/Δt

(초기 단계에서는 P 제어만 사용한다고 가정)
u = Kp × 1 = Kp  (양수 → 왼쪽 회전 명령)
```

#### STEP 4. 모터 속도 조정

```
기본 속도 = V_base
좌측 모터 속도 = V_base - u   (감속 → 왼쪽으로 틀림)
우측 모터 속도 = V_base + u   (증속 → 왼쪽으로 틀림)
```

#### STEP 5. 모터에 PWM 신호 출력

```
PWM_좌 = (V_base - u) / V_max × 255
PWM_우 = (V_base + u) / V_max × 255
→ 모터 드라이버(L298N 등)에 PWM 값 전달
```

#### STEP 6. 센서값 재확인 (루프)

```
다음 제어 주기(예: 10ms 후)에 다시 STEP 1부터 반복
→ 센서값이 000으로 돌아오면 정상 직진으로 복귀
```

### 제어 흐름도

```
[센서 읽기] → [패턴 판단] → [오차 계산]
                                  ↓
[모터 PWM 출력] ← [PID 계산] ← [오차 입력]
       ↓
[다음 주기 반복]
```

## 문제 2 — 엔코더 데이터로 이동 설명

### 스펙 확인

```
1펄스당 이동 거리 = π × 10cm / 30 = π/3 cm ≈ 1.047cm
휠베이스 L = 20cm
```

### 구간별 분석

#### 구간 A — 동일 100펄스 (직선 주행)

```
좌·우 바퀴 모두 100펄스
이동 거리 = 100 × π/3 = 100π/3 ≈ 104.72cm
→ 직선으로 약 104.72cm 전진
```

#### 구간 B — 좌바퀴 20펄스, 우바퀴 10펄스 (회전)

```
좌바퀴 이동 거리 d_L = 20 × π/3 = 20π/3 ≈ 20.94cm
우바퀴 이동 거리 d_R = 10 × π/3 = 10π/3 ≈ 10.47cm
```

**회전 방향:** 좌바퀴 > 우바퀴 → **오른쪽으로 회전**

**회전 각도:**

```
θ = (d_L - d_R) / L
  = (20π/3 - 10π/3) / 20
  = (10π/3) / 20
  = π/6 rad = 30°
```

**회전 반경 (로봇 중심 기준):**

```
R = L/2 × (d_L + d_R) / (d_L - d_R)
  = 10 × (30π/3) / (10π/3)
  = 10 × 3 = 30cm
```

**로봇 중심 이동 거리 (호의 길이):**

```
호 길이 = (d_L + d_R) / 2 = 5π ≈ 15.71cm
```

#### 구간 C — 동일 50펄스 (직선 주행)

```
좌·우 바퀴 모두 50펄스
이동 거리 = 50 × π/3 = 50π/3 ≈ 52.36cm
→ 직선으로 약 52.36cm 전진 (회전 후 새로운 방향 기준)
```

---

### 전체 이동 요약

| 구간           | 이동 유형 | 이동 거리                    | 비고                   |
| -------------- | --------- | ---------------------------- | ---------------------- |
| A              | 직선      | 100π/3 ≈**104.72cm** | 초기 방향 유지         |
| B              | 우회전    | 5π ≈**15.71cm** (호) | 30° 우회전, 반경 30cm |
| C              | 직선      | 50π/3 ≈**52.36cm**   | 회전 후 방향 유지      |
| **합계** | —        | **≈ 172.79cm**        | —                     |

### 이동 경로 도식

```
출발
  │ 104.72cm (직선)
  │
  ●── 회전 시작
   ╲  (30° 우회전, 반경 30cm)
    ╲
     ● ── 52.36cm (직선) ──► 도착
```
