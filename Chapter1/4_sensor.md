# 센서(Sensor) 및 IR 센서 조사 보고서

---

## 1. 센서의 정의 및 역할

**센서(Sensor)**란 물리적 환경(빛, 온도, 거리, 압력 등)의 변화를 감지하여 전기 신호로 변환하는 장치이다.

| 구분 | 내용                                                            |
| ---- | --------------------------------------------------------------- |
| 입력 | 물리량 (빛, 온도, 압력, 거리 등)                                |
| 출력 | 전기 신호 (전압, 전류, 디지털 값)                               |
| 역할 | 로봇이 주변 환경을 인식하고 적절히 반응할 수 있도록 데이터 제공 |

라인트레이서 로봇에서 센서는 **바닥의 경로(검은 선)를 인식**하여 주행 방향을 결정하는 핵심 부품이다.

## 2. 센서의 유형과 용도

| 유형                       | 감지 대상        | 용도 예시                   |
| -------------------------- | ---------------- | --------------------------- |
| **IR 센서** (적외선) | 빛 반사율        | 라인트레이싱, 장애물 감지   |
| **초음파 센서**      | 거리 (음파 반사) | 장애물 회피, 거리 측정      |
| **광 센서 (조도계)** | 빛의 밝기        | 밝기 감지, 라인 감지        |
| **온도 센서**        | 온도             | 과열 감지, 환경 모니터링    |
| **가속도 센서**      | 가속도, 기울기   | 자세 제어, 충격 감지        |
| **엔코더**           | 회전 각도/속도   | 모터 속도 피드백, 거리 계산 |
| **카메라**           | 영상             | 비전 기반 경로 인식         |

## 3. 센서 신호의 유형

### 3.1 아날로그 신호 (Analog Signal)

- 연속적인 값으로 출력 (예: 0V ~ 5V 사이 어떤 값이든 가능)
- 환경 변화에 따라 부드럽게 변화
- **예시:** 조도 센서, 온도 센서, IR 센서의 아날로그 출력
- 마이크로컨트롤러에서 읽으려면 **ADC(Analog-Digital Converter)** 필요

```
신호 전압
5V │    ╭─╮
   │   ╭╯  ╰╮
   │  ╭╯    ╰─╮
0V └─────────────▶ 시간
   (연속적으로 변화)
```

### 3.2 디지털 신호 (Digital Signal)

- HIGH(1) 또는 LOW(0) 두 가지 값만 출력
- 임계값(Threshold)을 기준으로 ON/OFF 판단
- **예시:** IR 센서의 디지털 출력, 버튼 입력
- 마이크로컨트롤러에서 직접 읽기 가능

```
신호
HIGH │▔▔▔▔╮    ╭▔▔▔
     │    │    │
LOW  │    ╰────╯
     └─────────────▶ 시간
   (HIGH 또는 LOW만 존재)
```

| 구분              | 아날로그            | 디지털        |
| ----------------- | ------------------- | ------------- |
| 출력값            | 연속적 범위         | HIGH / LOW    |
| 정보량            | 풍부                | 단순          |
| 처리              | ADC 필요            | 직접 처리     |
| 라인트레이싱 활용 | 선의 위치 정밀 파악 | 선 위/밖 판단 |

## 4. 센서의 해상도와 정확도

### 4.1 해상도 (Resolution)

https://www.youtube.com/watch?v=RN3B8HEk0-c

- 센서가 감지할 수 있는 **최소 변화량**
- ADC 비트 수에 따라 결정
  - 10bit ADC: 0~1023 단계 (2^10 = 1024)
  - 12bit ADC: 0~4095 단계 → 더 세밀한 측정 가능

### 4.2 정확도 (Accuracy)

- 측정값이 실제값에 **얼마나 가까운지**
- 온도, 전원 전압, 주변광 등 환경 요인의 영향을 받음
- 센서 캘리브레이션(보정)으로 정확도 향상 가능

| 개념   | 설명                                    |
| ------ | --------------------------------------- |
| 해상도 | 얼마나 세밀하게 구별하는가              |
| 정확도 | 실제값과 얼마나 일치하는가              |
| 정밀도 | 같은 조건에서 얼마나 일관된 값을 내는가 |

## 5. 응답 시간과 샘플링 속도

### 5.1 응답 시간 (Response Time)

- 환경 변화 후 센서가 안정된 출력값을 내기까지 걸리는 시간
- 응답 시간이 길면 빠른 변화를 놓칠 수 있음
- 라인트레이서에서는 **짧은 응답 시간**이 중요 (빠른 주행 시 경로 이탈 방지)

### 5.2 샘플링 속도 (Sampling Rate)

- 마이크로컨트롤러가 센서 값을 읽는 빈도 (단위: Hz, 회/초)
- 샘플링 속도가 낮으면 경로 변화를 늦게 감지 → 오작동 가능
- 나이퀴스트 정리: 신호 변화 주파수의 **최소 2배** 이상으로 샘플링 필요
  https://www.youtube.com/watch?v=iSl09GrYp-8

```
로봇 속도 0.5m/s, 선 폭 2cm 기준:
→ 선을 지나는 시간 ≈ 0.04초
→ 최소 샘플링 주기 < 0.02초 (50Hz 이상)
```

## 6. 센서의 노이즈와 필터링

### 6.1 노이즈 (Noise)

- 실제 신호에 섞이는 불필요한 신호
- 원인: 전원 불안정, 전자기 간섭, 주변광 변화, 진동

### 6.2 필터링 방법

| 필터 종류                | 방법                       | 특징                      |
| ------------------------ | -------------------------- | ------------------------- |
| **이동 평균 필터** | 최근 N개 값의 평균 사용    | 단순, 지연 발생           |
| **중앙값 필터**    | 최근 N개 값 중 중간값 사용 | 순간 노이즈 제거에 효과적 |
| **임계값 필터**    | 일정 범위 벗어난 값 무시   | 오측정 제거에 유리        |
| **저역통과 필터**  | 느린 변화만 통과시킴       | 고주파 노이즈 제거        |

## 7. IR 센서 (적외선 센서)

### 7.1 기본 원리

IR(Infrared) 센서는 **적외선(파장 780nm ~ 1mm)** 을 이용해 물체나 지면의 반사율을 감지한다.

```
      IR LED (송신)
          │
          ▼ 적외선 방출
    ┌─────────────┐
    │   지면 (바닥) │
    └─────────────┘
          │
          ▼ 반사된 적외선
      포토트랜지스터 (수신)
          │
          ▼ 전기 신호 출력
```

- **흰색 바닥**: 적외선을 많이 반사 → 수신 신호 강함 → HIGH (또는 낮은 아날로그 값)
- **검은색 선**: 적외선을 흡수 → 수신 신호 약함 → LOW (또는 높은 아날로그 값)

### 7.2 구성 요소

| 구성 요소                               | 역할                                           |
| --------------------------------------- | ---------------------------------------------- |
| **IR LED**                        | 적외선 빛을 방출 (송신부)                      |
| **포토트랜지스터 / 포토다이오드** | 반사된 적외선을 감지하여 전류로 변환 (수신부)  |
| **비교기 (Comparator)**           | 수신 신호를 임계값과 비교하여 디지털 출력 생성 |
| **가변저항 (Trimmer)**            | 임계값 조정 (감도 조절)                        |

### 7.3 라인 감지에 적합한 IR 센서 특성

| 특성        | 요구 조건                   | 이유                        |
| ----------- | --------------------------- | --------------------------- |
| 감지 거리   | 2~10mm                      | 바닥과의 적절한 간격 유지   |
| 응답 속도   | 빠를수록 좋음               | 고속 주행 시 경로 놓침 방지 |
| 주변광 영향 | 적을수록 좋음               | 실내 형광등 간섭 최소화     |
| 출력 형태   | 디지털 + 아날로그 모두 지원 | 단순 감지 및 정밀 위치 파악 |
| 크기        | 소형                        | 여러 개를 나란히 배치 가능  |

**대표 모듈:** TCRT5000, QTR-8RC, QTR-8A

### 7.4 IR 센서 신호 처리

**디지털 방식 (단순 라인 감지):**

```
센서값 = HIGH → 흰색 바닥 (선 없음)
센서값 = LOW  → 검은색 선 위
```

**아날로그 방식 (정밀 위치 파악):**

```
여러 개 센서의 아날로그 값으로 선의 중심 위치 계산

위치값 = (S1×0 + S2×1 + S3×2 + S4×3) / (S1+S2+S3+S4)
→ 값이 중앙(1.5)에서 벗어날수록 회전 필요
```

### 7.5 IR 센서 배치 시 유의사항

| 항목                   | 권장 사항                                                      |
| ---------------------- | -------------------------------------------------------------- |
| **센서 높이**    | 바닥으로부터 2~10mm (너무 높으면 감지 범위 넓어져 정밀도 저하) |
| **센서 간격**    | 선 폭(약 2~3cm)보다 좁게 배치 → 선 위치 정밀 감지 가능        |
| **센서 개수**    | 최소 3개 이상 권장 (좌/중/우 기본, 5개 이상이면 더 정밀)       |
| **주변광 차단**  | 센서 주변에 차광 구조 추가하면 외부 빛 간섭 감소               |
| **대칭 배치**    | 로봇 중심선 기준 좌우 대칭으로 배치 → 보정 알고리즘 단순화    |
| **캘리브레이션** | 사용 전 흰색/검은색 면에서 최대·최솟값 보정 필수              |

**권장 배치 예시 (5개 센서):**

```
     로봇 진행 방향
          ↑
   [S1][S2][S3][S4][S5]
    좌              우
    ↑               ↑
  외곽선          외곽선
  감지용          감지용
```
