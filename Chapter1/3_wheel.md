# 바퀴(Wheel) 조사 보고서

---

## 1. 바퀴의 구조와 재질

### 1.1 기본 구조

```
         타이어 (외부)
        ┌───────────┐
        │   림(Rim) │  ← 바퀴 본체 (원형 틀)
        │   ┌───┐   │
        │   │허브│   │  ← 중심축 연결부
        │   └───┘   │
        │  스포크    │  ← 허브와 림을 연결하는 살
        └───────────┘
              │
           차축(Axle)
```

| 구성 요소 | 역할 |
|-----------|------|
| **허브 (Hub)** | 바퀴 중심부, 차축과 연결되어 회전력 전달 |
| **스포크 (Spoke)** | 허브와 림을 연결, 하중 분산 |
| **림 (Rim)** | 바퀴 외곽 원형 틀, 타이어를 지지 |
| **타이어 (Tire)** | 지면과 직접 닿는 부분, 충격 흡수 및 마찰력 제공 |

### 1.2 재질

| 부위 | 주요 재질 | 특징 |
|------|-----------|------|
| 허브 / 림 | 알루미늄, 플라스틱, 강철 | 가볍고 강도 높음 |
| 타이어 | 고무 (천연/합성) | 탄성, 마찰력, 충격 흡수 |
| 소형 로봇용 바퀴 | ABS 플라스틱 + 고무 코팅 | 경량, 제작 용이 |

**라인트레이서 로봇의 경우:**
- 바퀴 재질이 주행 안정성에 직접 영향
- 고무 타이어: 마찰력 높아 미끄러짐 방지에 유리
- 단단한 플라스틱 바퀴: 마찰이 적어 빠르지만 미끄러질 수 있음

## 2. 마찰력과 관성력이 바퀴에 작용하는 방법

### 2.1 마찰력 (Friction Force)

마찰력은 두 물체가 접촉할 때 운동을 방해하는 힘이다.

#### 정지 마찰력 (Static Friction)

- 바퀴가 **아직 움직이지 않는 상태**에서 작용
- 외부 힘이 정지 마찰력보다 작으면 바퀴는 움직이지 않음
- 최대 정지 마찰력: `F = μs × N`
  - μs: 정지 마찰 계수
  - N: 수직 항력 (바퀴가 지면을 누르는 힘)

#### 동적 마찰력 (Kinetic Friction)

- 바퀴가 **이미 움직이는 상태**에서 작용
- 정지 마찰력보다 항상 작음 (`μk < μs`)
- 운동 방향과 반대로 작용하여 속도를 줄임

| 구분 | 발생 시점 | 크기 |
|------|-----------|------|
| 정지 마찰력 | 정지 → 움직임 시작 직전 | 더 큼 |
| 동적 마찰력 | 이미 움직이는 중 | 더 작음 |

**바퀴에서의 역할:**
- 모터가 바퀴를 돌리면 → 바퀴와 지면 사이 마찰력이 로봇을 앞으로 밀어줌
- 마찰력이 너무 작으면 바퀴가 헛돌고 로봇이 제대로 이동하지 못함

### 2.2 관성력 (Inertial Force)

- 물체가 현재 운동 상태를 유지하려는 성질 (뉴턴 제1법칙)
- 바퀴가 회전하기 시작할 때: 관성 때문에 더 큰 힘이 필요 (정지 → 출발)
- 바퀴가 회전 중일 때: 관성 때문에 쉽게 멈추지 않음 (진행 → 정지)

**로봇에 미치는 영향:**

| 상황 | 관성의 작용 |
|------|-------------|
| 출발 시 | 정지 관성으로 인해 출발에 더 많은 토크 필요 |
| 경로 변경 시 | 직진 관성으로 인해 회전이 늦어질 수 있음 |
| 정지 시 | 운동 관성으로 인해 즉시 멈추지 않고 밀림 발생 |

## 3. 구름 저항 (Rolling Resistance)

### 3.1 개념

바퀴가 지면 위를 **굴러갈 때** 발생하는 저항력.

- 미끄러지는 마찰(슬라이딩)과 다름
- 바퀴가 완전히 굴러도 여전히 저항이 존재함
- 구름 저항력: `F = Crr × N`
  - Crr: 구름 저항 계수 (재질과 지면에 따라 다름)
  - N: 수직 항력

### 3.2 구름 저항이 발생하는 이유

**① 변형 (Deformation)**
- 바퀴(타이어)와 지면이 접촉하면 미세하게 변형됨
- 변형되었다가 복원되는 과정에서 에너지 손실 발생

**② 접지면의 비대칭 압력 분포**
- 앞쪽 접지면: 바퀴가 밀어야 하므로 저항 발생
- 뒤쪽 접지면: 복원되며 밀어주지만 에너지가 완전히 회복되지 않음

**③ 미세 미끄러짐 (Micro-slip)**
- 접지면 내에서 일어나는 극미세한 슬립 현상

### 3.3 구름 저항에 영향을 주는 요인

| 요인 | 저항 증가 조건 | 저항 감소 조건 |
|------|----------------|----------------|
| 바퀴 재질 | 부드러운 고무 | 딱딱한 재질 |
| 지면 재질 | 부드럽고 거친 지면 | 딱딱하고 매끄러운 지면 |
| 바퀴 반지름 | 작은 바퀴 | 큰 바퀴 |
| 하중 | 무거울수록 증가 | 가벼울수록 감소 |
| 속도 | 고속에서 증가 경향 | — |

### 3.4 라인트레이서 로봇에의 적용

- 실내 평탄한 바닥에서는 구름 저항 계수(Crr)가 낮음 → 영향 적음
- 바퀴 크기가 클수록 구름 저항 유리 → 속도와 효율 향상
- 로봇 무게를 줄이면 수직 항력(N) 감소 → 구름 저항 감소
