# 모터(Motor) 조사 보고서

---

## 1. 모터의 기본 유형

### 1.1 DC 모터 (Direct Current Motor)

직류 전류를 이용해 회전력을 만드는 가장 기본적인 모터.

| 항목 | 내용 |
|------|------|
| 동작 원리 | 전류가 흐르는 코일이 자기장 안에서 힘을 받아 회전 |
| 특징 | 구조가 단순하고 가격이 저렴함 |
| 장점 | 속도 제어가 쉽고 높은 토크 출력 가능 |
| 단점 | 위치 제어가 어렵고 브러시 마모 발생 (브러시형의 경우) |
| 주요 용도 | 선풍기, 장난감, 라인트레이서 구동 바퀴 등 |

**종류:**
- **브러시 DC 모터**: 일반적인 DC 모터, 브러시로 전류 전달
- **브러시리스 DC 모터 (BLDC)**: 브러시 없이 전자적으로 제어, 수명이 길고 효율 높음

---

### 1.2 서보 모터 (Servo Motor)

위치, 속도, 토크를 정밀하게 제어할 수 있는 모터.

| 항목 | 내용 |
|------|------|
| 동작 원리 | 내부 피드백(엔코더)으로 현재 위치를 감지하고 목표 위치로 보정 |
| 특징 | 정밀한 각도 제어 가능 (일반적으로 0°~180°) |
| 장점 | 위치 제어 정확도 높음, 제어 신호가 단순(PWM) |
| 단점 | DC 모터보다 가격이 높음 |
| 주요 용도 | 로봇 팔, 조향 장치, 카메라 짐벌 등 |

---

## 2. 모터의 사양 항목

모터를 선택할 때 확인해야 할 주요 사양:

| 사양 항목 | 설명 |
|-----------|------|
| **정격 전압 (Voltage)** | 모터가 정상 동작하는 전압 (예: 5V, 12V) |
| **정격 전류 (Current)** | 정상 동작 시 소비 전류 |
| **토크 (Torque)** | 회전력의 크기 (단위: kg·cm, N·m) — 클수록 무거운 것을 움직임 |
| **RPM (분당 회전수)** | 1분당 몇 바퀴 도는지 — 클수록 빠름 |
| **효율 (Efficiency)** | 전기 에너지 → 기계 에너지 변환 비율 (%) |
| **무부하 전류** | 아무 하중 없이 회전할 때 소비 전류 |
| **정지 토크 (Stall Torque)** | 모터가 멈추기 직전 최대 토크 |

## 3. 모터의 제어 원리

### 3.1 회전 속도 제어

**PWM (Pulse Width Modulation, 펄스 폭 변조)** 방식을 사용한다.

- 전압을 켰다 껐다 하는 신호를 반복해서 평균 전압을 조절
- **듀티 사이클(Duty Cycle)**: 켜진 시간의 비율
  - 듀티 사이클 100% → 최대 속도
  - 듀티 사이클 50% → 절반 속도
  - 듀티 사이클 0% → 정지

```
PWM 50%:  ▔▔__▔▔__▔▔__  → 중간 속도
PWM 100%: ▔▔▔▔▔▔▔▔▔▔▔▔  → 최대 속도
PWM 0%:   ____________  → 정지
```

### 3.2 회전 방향 제어

전류의 방향을 바꾸면 회전 방향이 바뀐다.
→ **H-브리지 회로**를 이용해 전류 방향을 전환한다.

| IN1 | IN2 | 동작 |
|-----|-----|------|
| HIGH | LOW | 정방향 회전 |
| LOW | HIGH | 역방향 회전 |
| HIGH | HIGH | 브레이크 (정지) |
| LOW | LOW | 자유 회전 (coast) |

## 4. 모터 드라이버 (Motor Driver)

### 4.1 필요한 이유

마이크로컨트롤러(Arduino 등)의 출력 전류는 매우 작아(약 40mA) 모터를 직접 구동할 수 없다.
모터 드라이버가 **작은 제어 신호 → 큰 전류**로 증폭시켜 모터를 구동한다.

### 4.2 동작 원리

- H-브리지 회로를 내장하여 정방향/역방향 전환 가능
- PWM 신호를 받아 속도 조절

### 4.3 대표 모터 드라이버

| 모델 | 최대 전류 | 특징 |
|------|-----------|------|
| **L298N** | 2A | 가장 널리 쓰이는 듀얼 채널 드라이버, 2개 모터 동시 제어 |
| **L9110S** | 800mA | 소형 저전력용, 라인트레이서에 적합 |
| **TB6612FNG** | 1.2A | L298N보다 효율 좋고 발열 적음 |

### 4.4 배선 연결 (L298N 기준)

```
Arduino    →  L298N      →  DC 모터
---------      -------       --------
디지털 핀  →  IN1, IN2   →  방향 제어
PWM 핀    →  ENA         →  속도 제어
GND       →  GND
           →  모터 전원 (외부 배터리)
           →  OUT1, OUT2 →  모터 +/-
```
