# 피드백 제어 및 PID 제어 조사 보고서

---

## 1. 제어 시스템의 기초

**제어 시스템(Control System)** 이란 어떤 대상(플랜트)의 출력이 원하는 목표값을 따르도록 입력을 조절하는 시스템이다.

| 용어 | 의미 |
|------|------|
| **목표값 (Setpoint, r)** | 원하는 출력값 (예: 목표 속도 50cm/s) |
| **제어 대상 (Plant)** | 제어받는 시스템 (예: 모터, 로봇) |
| **출력값 (Output, y)** | 현재 실제 상태 (예: 현재 속도 45cm/s) |
| **오차 (Error, e)** | 목표값 - 출력값 (예: 5cm/s) |
| **제어 입력 (Control Input, u)** | 시스템에 가하는 신호 (예: PWM 값) |

---

## 2. 피드백 제어의 기본 개념

**피드백(Feedback)** 이란 출력값을 다시 입력 쪽으로 되돌려 오차를 줄이는 방식이다.

```
목표값(r) ──►[+]──► 제어기 ──► 플랜트 ──►── 출력(y)
             │-│                              │
             └──────────────────────────────┘
                        피드백
```

- **오차 e = r - y**
- 오차가 크면 → 더 강한 제어 입력
- 오차가 0이면 → 제어 완료

---

## 3. 피드백 제어의 기본 구조

### 3.1 오픈 루프 (Open Loop)

피드백 없이 입력만으로 제어.

```
목표값 ──► 제어기 ──► 플랜트 ──► 출력
```

- 단순하지만 외란(방해 요소)에 취약
- 예: 타이머로 모터를 3초 동안만 구동

### 3.2 클로즈드 루프 (Closed Loop)

출력을 측정하여 피드백으로 오차를 보정.

```
목표값 ──►[비교기]──► 제어기 ──► 플랜트 ──► 출력
              ↑                              │
              └─────────[센서]───────────────┘
```

- 외란이 발생해도 스스로 보정
- 라인트레이서: IR 센서로 경로 오차 감지 → 모터 속도 조절

| 구분 | 오픈 루프 | 클로즈드 루프 |
|------|-----------|---------------|
| 피드백 | 없음 | 있음 |
| 외란 대응 | 불가 | 자동 보정 |
| 구조 | 단순 | 복잡 |
| 정확도 | 낮음 | 높음 |

---

## 4. PID 제어의 개념

**PID 제어**는 오차(e)를 줄이기 위해 세 가지 항목(P·I·D)을 합산하여 제어 입력을 결정하는 방식이다.

```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
```

| 항목 | 이름 | 역할 |
|------|------|------|
| **P** | 비례 (Proportional) | 현재 오차에 비례하여 즉각 반응 |
| **I** | 적분 (Integral) | 과거 오차 누적을 제거 |
| **D** | 미분 (Derivative) | 오차 변화 속도를 예측하여 과도 반응 억제 |

---

## 5. P, I, D 각 항목의 역할과 수학적 표현

### 5.1 P 항 (비례 제어)

```
u_P = Kp × e(t)
```

- 오차가 클수록 강하게 반응
- Kp가 너무 크면 → **오버슈트(Overshoot)** 발생, 진동
- Kp가 너무 작으면 → 반응 느림, 정상 상태 오차 잔존

### 5.2 I 항 (적분 제어)

```
u_I = Ki × ∫e(t)dt
```

- 오차가 작게 지속될 때 누적하여 제거
- **정상 상태 오차(Steady-state error)** 제거에 효과적
- Ki가 너무 크면 → **적분 와인드업(Windup)** 으로 진동 발생

### 5.3 D 항 (미분 제어)

```
u_D = Kd × de(t)/dt
```

- 오차의 변화율(기울기)에 반응
- 오차가 빠르게 증가하면 미리 제동
- **오버슈트 억제**, 시스템 안정화
- 노이즈에 민감하므로 필터링과 함께 사용

### 각 항목의 영향 요약

| 증가시키면 | Kp | Ki | Kd |
|-----------|----|----|-----|
| 응답 속도 | 빨라짐 | 약간 빨라짐 | 느려짐 |
| 오버슈트 | 증가 | 증가 | 감소 |
| 정상 오차 | 감소 | 제거 | 변화 없음 |
| 안정성 | 저하 | 저하 | 향상 |
| 노이즈 민감도 | 변화 없음 | 변화 없음 | 증가 |

---

## 6. PID 튜닝

목표: 빠른 응답 + 오버슈트 최소 + 정상 상태 오차 = 0

### 6.1 수동 튜닝 순서 (Ziegler-Nichols 방식)

1. **Ki=0, Kd=0** 으로 시작
2. Kp를 천천히 증가 → 시스템이 지속 진동하는 임계 Kp(Ku) 찾기
3. 그 때의 진동 주기(Tu) 측정
4. 공식으로 초기값 계산:

| 제어기 | Kp | Ki | Kd |
|--------|-----|-----|-----|
| P만 | 0.5Ku | 0 | 0 |
| PI | 0.45Ku | 0.54Ku/Tu | 0 |
| PID | 0.6Ku | 1.2Ku/Tu | 3KuTu/40 |

### 6.2 라인트레이서에서의 PID 적용

```
오차(e) = 라인 중심 - 로봇 중심 위치 (IR 센서 값으로 계산)

좌측 모터 속도 = 기본속도 - PID출력
우측 모터 속도 = 기본속도 + PID출력
```

- e > 0 (로봇이 왼쪽으로 치우침) → 우측 모터 증속, 좌측 모터 감속
- e < 0 (로봇이 오른쪽으로 치우침) → 좌측 모터 증속, 우측 모터 감속
